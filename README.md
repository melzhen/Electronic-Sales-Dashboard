# Electronic-Sales-Dashboard
The goal of this project is to create a dashboard using Power BI to analyze customer purchase behavior by identifying trends in sales revenue, product demand, and member loyalty engagement. By building a dashboard, we are able to provide actionable insights to increase revenue, reduce order cancellations, and improve overall customer satisfaction.

Dataset used: https://www.kaggle.com/datasets/cameronseamons/electronic-sales-sep2023-sep2024 

### **About Dataset**

The data used to create this dashboard was extracted from the Customer purchase behavior - Electronic Sales Data dataset via Kaggle. All data was auto-generated for analysis purposes and does not represent real-world transactions or use actual customer data.This dataset contains exactly 20,000 rows of sales transaction data for an unspecified electronics company over a one-year period, from September 2023 to September 2024. There are no duplicate rows or columns. Key features to be used for the dashboard include: Customer ID, Loyalty Member, Product Type, Rating, Order Status, Total Price, Quantity, and Purchase Date.

**Dashboard Highlights**
- Total Revenue made from completed orders $42.63M
- Total number of distinct customers 12,136

- Total units sold 73.7K
- Customer satisfaction score (CSAT) vs. hypothetical goal (80%) 52.35%

**Visualizations**

- Gauge meter to highlight the actual CSAT score to the hypothetical target
- Donut chart for comparison between the number of completed orders vs. canceled orders
- Donut chart for comparison of customer distribution by loyalty status 
- Line chart comparing completed and cancelled orders over time
- Bar chart of revenue by product category
- Bar chart of revenue accumulated over time (MoM)
