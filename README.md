# Electronic-Sales-Dashboard
The goal of this project is to create a dashboard using Power BI to analyze customer purchase behavior by identifying trends in sales revenue, product demand, and member loyalty engagement. By building a dashboard, we are able to provide actionable insights to increase revenue, reduce order cancellations, and improve overall customer satisfaction.

Dataset used: https://www.kaggle.com/datasets/cameronseamons/electronic-sales-sep2023-sep2024
